# 代码文本类

该类别的特征：与文本类类似，区别在于所涉及的文本为代码/代码块形式

## region2indent

```js
id: "region2indent",
name: "代码注释转缩进",
detail: "代码块注释转缩进 (识别 `//` 和 `#` 的region注释对)，通常配合code2list使用。默认补充两缩进",
process_param: ABConvert_IOEnum.text,
process_return: ABConvert_IOEnum.text,
```

id: "mdit2code",
